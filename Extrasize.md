## Installation

Get the project running locally in your machine.

1.  Unzip the project folder



2.  ```
    cd Server
    ```

3.  Install the dependencies

    ```
    npm install
    ```

4.  Run the server

    ```
    npm run dev
    ```

5.  ```
    cd client
    ```

6.  Install the dependencies

    ```
    npm install
    ```

7.  Run the client

    ```
    npm run dev
    ```
